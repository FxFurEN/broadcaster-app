## broadcaster-app

Simple application, made as a large laboratory work for the subject "Database and DBMS"

Using:
- BoldReports
- DocumentFormat
- WPF
- 6.0 Net
- HandyControls
- LiveCharts
- Visual Studio 2022 
- MSSQL
- Report Builder
- SQL Server Managment Studio
